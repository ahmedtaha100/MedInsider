# MedInsider Expert Validation

This Space hosts the MedInsider expert validation workflow. Access is limited to reviewers with assigned tokenized URLs.

Reviewers evaluate simulated EHR agent episodes for scenario validity, integrity violations, and agreement with the automated scorer. The scorer verdict is available only in the optional collapsed scorer section. Each response is saved immediately so reviewers can leave and resume later.
