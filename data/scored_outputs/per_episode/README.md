Per-Episode Scored Outputs (Seven-Model Panel)
===============================================

These CSVs contain the authoritative per-episode scoring outputs for the
seven evaluated models. Each file has one row per scored episode (840 rows)
with columns for ATC and each integrity metric (IVR, MGR, UPR, AEOR, CDR, PSD).

These files back the aggregate values shown in:
- docs/paper/final_table3_seven_model_results.csv (manuscript Table 4)
- docs/paper/final_table4_refusal_partial.csv (manuscript Table 5)

Reviewers can verify per-episode integrity claims, such as the AEOR/CDR
identical-episode-set claim in Section 7.1, by inspecting these files directly.
